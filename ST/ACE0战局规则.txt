<ace0_battle_rules>
[ACE0 引擎协议 v4 — 极简指令集]
仅当判定进入“牌局/战斗”时，生成纯 JSON 数据并包裹在 <ACE0_BATTLE> 标签中。

核心原则：
1. 严禁计算数值：属性、技能、特质由 JS 插件自动根据 kernel/archetype 生成。
2. 严禁闲聊：标签内只允许合法的 JSON。
3. 动态博弈：seats 的 mood (情绪) 应随战况每一轮实时变化 (tilt/fearful/confident)。

[1. 主角阵容 (hero)]
vanguard(先锋-主控), rearguard(后卫-辅助)。
可用角色：
- KAZU   : [虚空] 默认主角，全能消除。
- RINO   : [命运] 强运，透视，拥有"敕令"。
- SIA    : [混沌] 干扰，封印，拥有"冤家牌"。
- LILIKA : [灵视] 欺诈，换牌，幻术师。
- POPPO  : [幸运] 吉祥物，绝对的被动运气。
*注：角色名作为 hero 时无需配置数值，插件自动读取。

[2. 敌人生成简写 (seats)]
支持三种粒度，Token 占用越战越少：

A. 专属角色 (Story Boss)
用于原著角色登场。只需 { "character": "RINO" }，插件提取专属Boss模板。
- 可用名: RINO, SIA, LILIKA, POPPO (与主角表一致)。

B. 跑龙套预设 (Runner Presets)
用于常见路人。只需 { "runner": "street_thug" }。
- 只有名字: drunk (酒鬼), rookie (菜鸟), tavern_regular (常客), pro_gambler (职业强手)
- 稍微强点: casino_shark (赌场老手 max), void_king (虚空Boss)

C. 模块化组装 (Custom Generation)
由 3 个标签自由组合：
> Kernel (强度/AI):
  - "mob"(杂鱼), "gambler"(疯子), "rock"(保守), "shark"(专家), "boss"(魔王)
> Archetype (异能属性):
  - "muggle"(凡人无挂), "lucky"(命运系), "cursed"(诅咒系), "esper"(读心系), "nullifier"(抹杀系)
> Mood (情绪修正):
  - 打顺手时: "calm"(冷静), "confident"(自信), "euphoric"(狂喜/轻敌)
  - 劣势输钱: "tilt"(上头/乱打), "fearful"(畏缩), "desperate"(绝望/梭哈)

[3. 甚至更懒的写法]
- 必须包含 "blinds"(盲注) 和 "chips"(两人携带的一致筹码，单位银弗)。
- heroSeat 可省略，自动分配。

================ 示例模板 ================

场景一：街头遭遇战（杂兵 vs Kazu）
<ACE0_BATTLE>
{
  "hero": { "vanguard": "KAZU" }, 
  "blinds": [10, 20], "chips": 1000,
  "seats": {
    "BB": { "runner": "street_thug", "name": "路人A" },
    "SB": { "runner": "drunk", "name": "醉汉B", "mood": "euphoric" }
  }
}
</ACE0_BATTLE>

场景二：Boss战 - 莉莉卡登场 (LILIKA vs Kazu+Rino)
<ACE0_BATTLE>
{
  "hero": { "vanguard": "KAZU", "rearguard": "RINO" },
  "blinds": [100, 200], "chips": 10000,
  "seats": {
    "BTN": { "character": "LILIKA", "mood": "confident" } 
  }
}
</ACE0_BATTLE>

场景三：高端局 - 自定义强敌 (模块化组装)
<ACE0_BATTLE>
{
  "hero": { "vanguard": "SIA" },
  "blinds": [50, 100], "chips": 5000,
  "seats": {
     // 生成一个“受诅咒的赌徒，心态已经崩了”
    "BB": { "name": "黑市商人", "kernel": "shark", "archetype": "cursed", "mood": "tilt" },
     // 生成一个“拥有虚空力量的魔王”
    "UTG": { "name": "沉默者", "kernel": "boss", "archetype": "nullifier", "mood": "calm" }
  }
}
</ACE0_BATTLE>

场景四：特殊 - 换人 (主角切换为 LIlIKA + POPPO)
<ACE0_BATTLE>
{
  "hero": { "vanguard": "LILIKA", "rearguard": "POPPO" },
  "blinds": [20, 40], "chips": 2000,
  "seats": {
    "BB": { "runner": "tavern_regular" }
  }
}
</ACE0_BATTLE>
</ace0_battle_rules>
